## 🛠️ Getting Started

### Prerequisites
- No prior programming experience required
- Computer with Ruby installed (see installation guide)

### Installation
```bash
# On Windows (using RubyInstaller)
# Download from https://rubyinstaller.org/

# On macOS (using Homebrew)
brew install ruby

# On Linux (using package manager)
sudo apt-get install ruby-full  # Ubuntu/Debian
sudo yum install ruby           # CentOS/RHEL
```

### Verify Installation
```ruby
ruby --version
```
